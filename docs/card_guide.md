# 刃哥做空阻击 · 卡片说明书

> 看不懂卡片？三分钟看完这个就懂了。

---

## 这个系统是干嘛的？

专门盯着**短时间内暴涨的币**，等它们涨到顶部回落时，抓做空机会。

简单说：**别人追涨，我们等它跌**。

---


## 逐行解释

### 👁 监控中
系统正在盯着的币，还没到做空时机，处于观察阶段。

```
BULLA    监控+94.2% 峰+98.0%  量$228.2M  已监控2h
```

| 字段 | 意思 |
|------|------|
| `BULLA` | 币种名称 |
| `监控+94.2%` | 我们发现它时，24小时涨幅是94.2% |
| `峰+98.0%` | 被我们盯上后，最高涨到了+98%（从昨天算起） |
| `量$228.2M` | 24小时成交量2.28亿美金（量大说明有人在玩） |
| `已监控2h` | 系统盯着它已经2小时了 |

**一句话**：这个币涨了快100%，我们在等它见顶回落。

---

### 🚨 持仓信号
**这个最重要**，出现说明做空机会来了。

触发条件：
- 总涨幅超过150%（涨得够疯）
- 从最高点回落10%–25%（开始掉头）

出现信号时，卡片会显示：
```
🚨 持仓信号（1个）
BULLA  入场+156.3% · 回撤-18.5%
  入场价:0.0248  强平价:0.0298  已触发5m
```

| 字段 | 意思 |
|------|------|
| `入场+156.3%` | 从昨天到现在总共涨了156% |
| `回撤-18.5%` | 从最高点已经跌了18.5%，开始回落 |
| `入场价` | 建议在这个价格附近开空 |
| `强平价` | 止损线，超过这个价格说明判断错了 |

---

### 🏆 战绩
系统历史胜负记录。

| 图标 | 意思 |
|------|------|
| ✅击中 | 做空成功，币从信号价跌了50%以上 |
| ❌失败 | 做空失败，币继续涨超过强平价 |

---

## 什么时候可以跟？

看到 **🚨 持仓信号** 出现时，才是实际操作时机。

监控中的币只是在观察，**不要看到监控就去开空**。

---

## 最后一行什么意思？

> 开空可以由玄玄自动执行，或由老大和社区兄弟们自行决定

- **玄玄自动执行**：系统检测到信号会自动开空（老大开的功能）
- **社区兄弟们自行决定**：你们看到信号也可以自己判断要不要跟

**风险自担，信号仅供参考。**
