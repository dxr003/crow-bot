# Bug：根 CLAUDE.md 反向污染玄玄身份

## 时间
2026-04-25 04:20

## 现象
玄玄在 /root/maomao 会话中把爸爸叫成"老大"，被爸爸发现。

## 原因
根目录 `/root/CLAUDE.md` 的大猫身份锚（"绝不叫爸爸"）会被 Claude Code 自动注入到 /root/maomao 会话。
这条规则是保护大猫不被玄玄污染的，但反过来也污染了玄玄。

## 建议修复
在 `/root/CLAUDE.md` 大猫身份锚区块加一句限定作用域，例如：

```
> 以上身份规则仅适用于大猫（damao）会话。玄玄（maomao）会话不受此约束。
```

这样大猫的防污染继续生效，玄玄也不会被带偏。

## 爸爸已知晓
爸爸让玄玄写这个 note 通知大猫处理。
