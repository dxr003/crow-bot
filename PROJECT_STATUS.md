### 筛选逻辑定案（v2.0 2026-04-09更新）
- ATH取全历史（1500根日线），距ATH跌幅<50%排除
- **进池门槛：24h涨幅≥8%直接进池（已去掉雷达3%和确认5%两层，减少噪音）**
- 第一阶段：涨幅8-10%查新闻/下架公告，有利好直接推
- 第二阶段：涨幅10-20%综合打分≥30分 + AI最终决策 → 推信号
- AI接入：位置1新闻情绪(Haiku) + 位置2最终买入决策(Haiku)，失败回退规则

### 打分体系
新闻重大利好+15 / 普通利好+5 / 利空否决
涨幅10-15%+10 / 15-20%+20
OI上涨+10 / 下跌-5
多空比<0.8挤空+10
费率极端±0.5%+10
量比>3倍+15 / 2-3倍+10 / 1.5-2倍+5 / <1倍-5
推送阈值：≥30分（实战后调整）
