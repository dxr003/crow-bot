
## 做多阻击（bull_sniper）

捕捉合约市场短线爆发机会，两阶段过滤，综合打分推信号。

### 筛选过滤
- 上线 < 30天排除
- 24h涨幅已 > 30%排除
- 日成交额 < 500万U排除
- 距历史ATH跌幅 < 50%排除（未腰斩不碰，全历史1500根日线）

### 触发机制
```
雷达：1分钟涨≥3%
    ↓
确认进池：3分钟内5分钟累计涨≥5%
    ↓
第一阶段：涨幅≥8% → 查新闻+下架公告 → 有利好直接推
    ↓
第二阶段：涨幅10-20% → 综合打分≥30分推信号
```

### 打分体系

| 指标 | 条件 | 分值 |
|------|------|------|
| 新闻 | 重大利好 | +15 |
| 新闻 | 普通利好 | +5 |
| 新闻 | 利空 | 直接否决 |
| 涨幅 | 10-15% | +10 |
| 涨幅 | 15-20% | +20 |
| OI | 同步上涨 | +10 |
| OI | 下跌 | -5 |
| 多空比 | < 0.8空头占多 | +10 |
| 费率 | ≤-0.5%或≥+0.5% | +10 |
| 量比 | > 3倍 | +15 |
| 量比 | 2-3倍 | +10 |
| 量比 | 1.5-2倍 | +5 |
| 量比 | < 1倍 | -5 |

推送阈值：≥30分（实战后调整）

### 新闻源
- 主力：Tavily（限定CoinDesk/CoinTelegraph/TheBlock等专业加密媒体）
- 备用：Google News RSS（Tavily连续失败3次自动切换）
- 下架通道：币安exchangeInfo原生API（查合约status），独立不走评分

### 推送
- 标题：小刃做多阻击信号预警
- 触发立即推 + 整点XX:00状态卡片（含评分明细/新闻摘要）
- 健康播报：每小时私信乌鸦（不进群）

## 做空阻击（short_attack）

扫描暴涨榜，捕捉拉盘见顶后的做空机会。已封板。

| 参数 | 值 |
|------|-----|
| 监控门槛 | 24h涨幅≥80% |
| 信号触发 | 从发现价涨≥150% 且回撤≥10% |
| 强平价 | 信号价×120% |
| 卡片标题 | 刃哥做空阻击 |
| 整点推送 | 每小时:01 |
