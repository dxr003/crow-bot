## 2026-04-07 · 做空阻击上线 + 移动止盈/滚仓封板 + 自动交易全链路

### 自动交易全链路
信号触发 → 自动开空 → 绑定移动止盈 → 登记滚仓 → 私信回执

| 参数 | 值 |
|------|-----|
| 金额 | 100U |
| 杠杆 | 10x |
| 保证金 | 逐仓 |
| 开单方式 | 暗单（智能拆单） |
| 移动止盈 | 自动激活，默认40%阈值 |
| 滚仓 | 登记名单，浮盈≥50%自动执行 |

**AUTO_TRADE 开关**（`/root/short_attack/.env`）：
- `AUTO_TRADE=true` — 信号触发自动开空
- `AUTO_TRADE=false` — 只推通知不开单
