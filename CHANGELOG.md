## v3.1.0 · 2026-04-12

### 观察池逻辑重构（重大升级）
- 池内去掉所有1h依赖（退出/触发/通道分流），进池后1h使命完成
- 去掉5m≥5%前置门槛，池内定期打分，无固定时间窗口卡点
- 去掉30分钟超时退出，零成本观察不主动踢
- 池满淘汰最弱（进池后涨幅最低），替代淘汰最老
- 打分冷却去掉5分钟限制，价格变化≥1%即重新打分
- watchpool_max 10 → 30
- 卡片显示"池内+X%"（进池后涨幅）替代"进池+X%"（1h）

### OI变化实装
- get_oi_change() 从占位return 0改为币安原生openInterest API
- 5分钟缓存对比，OI评分两档（+5/+10/-5）正式生效

### 评分阈值调整
- signal_threshold 28 → 25（配合无门槛打分，慢涨币不被卡）
- analyzer打分通道去掉gain_pct范围限制（原10-35%）

### 仓位与止损调整
- position_usd 20U → 50U
- 止损保证金亏损 15% → 30%（5x杠杆=价格跌6%触发，亏15U）

### 快照推送
- status_push.py 改用 PUSH_BOT_TOKEN（播报bot），不再污染玄玄对话框

**设计理念：** 进池零成本，不应有时间/涨幅门槛限制观察。池内只靠评分体系判断买入，爆发型和慢涨型币都能被捕捉。

## v3.0.0 · 2026-04-10

### 做多阻击 scanner v3.0（重大重写）
从 v2.0 单层架构重写为三层过滤：

**第一层（全市场30秒扫描）— 基础过滤：**
- 24h涨幅 > 40% 排除
- 日成交额 < 2000万U 排除
- 上线天数不足排除
- 距ATH跌幅不足排除

**第二层（候选池→观察池）— 动量确认：**
- 1小时涨幅 ≥ 5%（新增：替代旧版24h涨幅8%入池）
- OI ≥ 500万U（新增）
- 量比 ≥ 2倍（新增）

**第三层（观察池触发）— 买入判断：**
- 1h涨幅在区间内 AND 5m ≥ 5% → 触发分析+买入
- 超出区间或超时30分钟 → 退出

**新增防刷机制：**
- 观察池退出冷却30分钟（解决3000+/小时重复进池问题）
- AI分析冷却5分钟（解决同币种重复调用Haiku问题）

### 移动止盈 v4.0（重大升级）
- 计算方式从价格跌幅改为**保证金收益率**：价格变动% × 杠杆
- 利润回撤替代价格回撤：(峰值浮盈 - 当前浮盈) / 峰值浮盈
- 激活阈值：保证金浮盈 ≥ 50%（旧版40%基于价格）
- 回撤触发：利润回撤 ≥ 30%，容错±3%
- 杠杆自动从 notional/initialMargin 推导（API不返回leverage字段）
- cron 频率从5分钟改为1分钟（高杠杆下价格波动影响放大）
- 实盘验证：RAVE 12x杠杆，峰值+80.2%，回撤42.2%成功触发

**升级原因：** RAVE 价格跌5.6%，旧版算浮盈5.6%不触发40%阈值。实际12x杠杆下保证金浮盈67.2%，应当触发。

### 做空阻击 Bug 修复
- 修复 `from trader.precision import fix_price` → `from trader.exchange import fix_price`（precision.py不存在）
- 修复卡片"阻击信号"和"没有发现买入信号"矛盾显示
- 合并 pending/positions 为统一"做空中"区域
- 空状态文案改为"暂无狙击目标"
